# 🌐 Aura — The AI Browser Companion

> *Your browser finally knows you.*

Aura is a Chrome Extension that replaces your new tab page with an AI-powered
personal dashboard. It learns your interests, reads your mood, curates your
world, and actually cares about your wellbeing.

Built with the Anthropic Claude API. Built for humans.

---

## ✨ Features

### 🏠 Smart New Tab Dashboard
Your new tab is no longer blank. It's your world — a beautiful, dark-themed
dashboard with your curated feed, AI search, stats, and more. Loads instantly.

### 🧠 Taste Engine
Aura silently learns what you love by tracking which sites you visit and how
long you stay. No form to fill out. No stars to rate. It just learns.

### 😶‍🌫️ Mood Detection
A quick mood check-in every session. You pick how you feel — Aura reshuffles
your entire feed to match. Tired? Lighter content. Hyped? Deep reads.

### 📰 Unified AI Feed
One feed. All your interests. Anime news, tech articles, sports headlines —
pulled from multiple sources, deduplicated, and curated by Claude with a
personal "why you'd like this" caption on every card.

### 🔍 AI-Powered Search
Type anything in the Aura search bar. Claude answers directly — not a list of
links. Ask for recommendations, get answers, find things faster.

### 🕐 Wellness System
Aura tracks how long you've been browsing and nudges you to rest. Not
annoyingly — smartly. Claude generates a fresh, never-repeated micro-activity
every time you need a break.

### 🎮 Mini Games
Hit your break threshold and a 2-minute mini game launches — trivia questions
generated by Claude based on *your* actual browsing history. Always relevant.
Never generic.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- Chrome or any Chromium-based browser (Brave, Edge, Arc)
- An [Anthropic API key](https://console.anthropic.com/)
- A [NewsAPI key](https://newsapi.org/) (free tier works)

### Installation

```bash
# 1. Clone the repo
git clone https://github.com/your-username/aura-extension
cd aura-extension

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env.local
# Add your API keys to .env.local

# 4. Build the extension
npm run build

# 5. Load into Chrome
# → Go to chrome://extensions
# → Enable "Developer Mode" (top right toggle)
# → Click "Load unpacked"
# → Select the dist/ folder
```

### Environment Variables

Create a `.env.local` file:

```env
VITE_ANTHROPIC_API_KEY=your_anthropic_api_key
VITE_NEWS_API_KEY=your_newsapi_key
```

## 🤖 How Claude Powers Aura

Aura uses Claude at 4 key moments:

| Moment | What Claude Does |
|--------|-----------------|
| Feed loads | Ranks articles + writes personal captions for each card |
| User searches | Answers directly with taste-aware recommendations |
| Break time | Generates a fresh, personalized wellness micro-activity |
| Mini game | Creates trivia questions from your actual browsing history |

- **Motion:** Smooth 300ms transitions, staggered feed animations, liquid
  mood reshuffles

## 🔒 Privacy

Aura is **fully local-first.**

- All your browsing data stays on your device in `chrome.storage.local`
- No account required, no backend, no database
- The only external calls are:
  - Anthropic Claude API (to generate captions, answers, trivia, break prompts)
  - NewsAPI / Jikan / HackerNews (to fetch article content)
- Your taste profile and history are **never sent anywhere** except as
  context to Claude for curation
